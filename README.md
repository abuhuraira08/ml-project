# End-to-End Machine Learning Application

## Fetal Health Prediction System

This project is an end-to-end machine learning application developed using Python, Scikit-learn, and Streamlit.

The model predicts fetal health condition based on cardiotocography (CTG) data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Features
- Data preprocessing
- Model training and evaluation
- Real-time prediction
- Streamlit web interface
- Model deployment
